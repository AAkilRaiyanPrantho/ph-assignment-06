Var Let Const

-> Var, Let and Const, all three of them are used to declare variables
-> But Let and Const are Block scope variables but Var creates Global scope variable
-> Var declaration gets Hoised but Let and Const declaration does not
-> Var lets you declare the same variable multiple times. But Let and Const let you declare a variable once and also, Let let's you update or re assign the same variable multiple times, but Const doesn't let you re-assign a value more than once

Null & Undefined

-> NULL and Undefined can be considered as an exam paper
-> Meaning, if one does not give any input, then it is an Undefined value. Which can be compared to not submitting the exam paper at all
-> On the other hand, NULL means that there is an input but the input's value is Empty. Meaning that the exam paper is submitted but nothing is written in it.

REST API

->REST is a set of architectural constraints, not a protocol or a standard. API developers can implement REST in a variety of ways.
-> Request of a client made by REST API transfers a representation of the state of the resource to the requester or endpoint.
-> Then this information, or representation, is delivered in one of several formats via HTTP: JSON (Javascript Object Notation), HTML, XLT, Python, PHP, or plain text. JSON.
